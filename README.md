# Biblioteca Online

## Overview

Biblioteca Online is a Java application designed to manage a digital library system. The project allows users to organize, store, search, and manage books in an efficient way while demonstrating fundamental Object-Oriented Programming concepts and data management techniques.

The application provides a simple solution for handling book records, making it easier to keep track of available resources within a library environment.

## Features

* Add new books
* View all books
* Search books by title or author
* Update book information
* Remove books from the library
* Manage book records efficiently
* User-friendly interface
* CRUD operations for library management

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Collections Framework
* File Handling
* IntelliJ IDEA
* Maven

## Main Functionalities

### Add Books

Users can add new books by providing details such as:

* Title
* Author
* Publication Year
* Category

### View Library

Displays all books currently stored in the system.

### Search Books

Allows users to quickly locate books using specific criteria.

### Update Book Information

Book details can be modified whenever necessary.

### Delete Books

Books can be permanently removed from the library.

## Installation

### Clone the Repository

```bash
git clone https://github.com/damaris-d23/BIBLIOTECA-ONLINE.git
```

### Open the Project

Open the project using IntelliJ IDEA or any Java IDE.

### Build the Project

```bash
mvn clean install
```

## Running the Application

Run the main class:

```text
Main.java
```

The application will start and display the available library management options.
